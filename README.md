# Dia
**Fast Brief:** Lightweight diagramming tool — build precise flowcharts, UML, and network maps with ease.

## Overview

Dia is a straightforward GTK-based diagram editor inspired by Microsoft Visio but entirely free and open source. It provides a clean canvas and a rich library of pre-made shapes spanning flowcharts, UML class and sequence diagrams, network topologies, electrical circuits, and entity-relationship models.

The application stores diagrams in a compact custom XML format and exports to EPS, SVG, PNG, and other standard formats. Its modular shape architecture allows the community to contribute new sheet libraries, and the multi-window interface keeps tool palettes accessible without cluttering the drawing area.

## Getting Started

Install Dia from your distribution's package manager or download the Windows installer. Launch Dia and select a shape sheet from the dropdown toolbox. Click a shape in the palette, then click the canvas to place it. Connect shapes by selecting the line tool and dragging between connection points on two objects.

## Everyday Use

1. Open Dia and choose the appropriate shape sheet for your diagram type.
2. Place shapes on the canvas with single clicks or click-and-drag for sizing.
3. Draw connectors between shapes using the line or polyline tools.
4. Double-click any shape to add labels, attributes, and descriptive text.
5. Group related elements to move and style them as a single unit.
6. Export via File > Export to produce crisp SVG or PNG for your documents.

## Troubleshooting

1. **Shapes don't appear in the toolbox:** Verify the shape sheet directory path in Edit > Preferences; reinstall the dia-shapes package if sheets are missing from the system library.
2. **Connectors route awkwardly:** Switch from straight lines to zigzag or bezier line types in the line properties; manually add midpoints by right-clicking a line and selecting Add Corner.
3. **Export produces blurry PNG:** Increase the export resolution in File > Export > PNG Options and ensure anti-aliasing is enabled.
4. **Diagrams look cramped on print:** Adjust the page setup scaling percentage in File > Page Setup and use the Print Preview to verify layout before printing.
5. **UML attribute fields truncated:** Widen the class shape by dragging its resize handles; long attribute lists may also benefit from reducing font size in the object properties.
